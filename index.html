Hepadnaviridae-GLUE

A GLUE project for the Hepadnaviridae.

Description

This is Hepadnaviridae-GLUE: a GLUE project for the hepadnaviruses (family Hepadnaviridae).

GLUE is an open source, data-centric bioinformatics environment specialised for developing virus genome data resources (VGDR).

This GLUE project contains reference information for the hepadnavirus family, including:

  1. A set of hepadnavirus reference sequences linked to auxiliary data.
  2. A comprehensive list of hepadnavirus genome features and their locations within hepadnavirus reference sequences.
  3. Alignments of hepadnavirus reference sequences arranged hierarchically by clade.

Who can use this resource, and for what?

Hepadnaviridae-GLUE can be used as a straightforward data repository, with no requirement for use of the GLUE software framework.

In addition, the Hepadnaviridae-GLUE project can be developed within the GLUE framework by extending the core dataset with new data and functionality. So far we have used Hepadnaviridae-GLUE to develop the following GLUE extension projects.

  • Hepadnaviridae-EVE - a GLUE project for endogenous hepadnaviral elements.

Installation

GLUE projects can be installed on computers running Windows, MacOSX or Linux, as follows:

  1. Install GLUE, based on the GLUE installation instructions.
  2. Clone the Hepadnaviridae-GLUE repository into your `gluetools/projects` directory.
  3. Navigate to the `gluetools/projects/Hepadnaviridae-GLUE` directory on the command line, initiate GLUE, and build the project by issuing the following command in the GLUE shell:
Mode path: /
GLUE> run file hepadnaviridae.glue
