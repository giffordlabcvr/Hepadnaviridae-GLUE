CRESS-GLUE

A GLUE project for circular Rep-encoding single-stranded DNA (CRESS DNA) viruses.

CRESS virus sequence data

Genome-length master reference sequences of representative CRESS DNA virus species, linked to auxiliary data in tabular format. These XML-formatted files are downloaded directly from NCBI and are uniquely identified within this project by their GenBank accession numbers.
